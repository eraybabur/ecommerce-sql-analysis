# ecommerce-sql-analysis
 E-commerce Sales SQL Analysis

 Project Overview
This project analyzes an e-commerce sales dataset using SQLite
The goal is to calculate key business metrics and evaluate category-level performance.

 Dataset
- 3,500 rows
- Columns: Order Date, Product Name, Category, Region, Quantity, Sales, Profit
- Imported from CSV into SQLite

Key Analysis

1. Overall KPI
- Total Sales
- Total Profit
- Overall Profit Margin

  2. Category-Level Analysis
- Total Sales by Category
- Total Profit by Category
- Profit Margin by Category
- Sorted by highest margin

Tools Used
- DB Browser for SQLite


Key Learnings
- Aggregate functions (SUM)
- GROUP BY
- ORDER BY
- Aliasing with AS
- ROUND function
